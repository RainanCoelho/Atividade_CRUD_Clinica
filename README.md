# Projeto Clínica Skrub
 
Esse é um sistema CRUD que tem a finalidade de fazer uma junção entre o banco de dados feito em SQL e o código que faz as aplicações em JAVA. 

---
# Tecnologias Utilizadas:
- Linguagem JAVA
- Biblioteca JDBC do Postgres
- E o SQL foi feito no POSTGRES
---

# Instrução de Execução:
  Cada classe vai possuir os mesmos métodos, ou seja, a mesma estrutura de código, alterando apenas os valores que irão entrar em cada método, já que são classes diferentes e possuem atributos diferentes.
  E toda vez que é chamada uma função para um objeto, se tem que escolher quais atributos serão mudados, isso variando de tabela para tabela
