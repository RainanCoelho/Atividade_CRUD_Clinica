# Projeto Clínica Estudo
 
Esse é um sistema CRUD que tem a finalidade de fazer uma junção entre o banco de dados feito em SQL e o código que faz as aplicações em JAVA. Usado para fazer uma gestão de aluno, turma e curso em um aplicativo de uma app de estudo

---
# Estrutura do Repositório:
- diagrama: Modelo Entidade-Relacionamento conceitual
- ddl: Scripts de criação (CREATE TABLE, Constraints).
- dml: Scripts de manipulação (INSERT, UPDATE, DELETE).
- dql: Scripts de consulta (SELECT, JOINs, Filtros).
- projeto: Código-fonte feito na linguagem JAVA

---
# Como executar o projeto:
- Importe o projeto na sua IDE (VS Code, IntelliJ ou Eclipse).
- Certifique-se de ter o driver JDBC no classpath.
- Execute a classe Main.java.

---
# Tecnologias Utilizadas:
- Linguagem JAVA
- Biblioteca JDBC do Postgres
- E o SQL foi feito no POSTGRES
